# Roblox Income Project

## Opportunity
- $922.8M annual creator payouts (2024-2025)
- Top 1,000 creators average $1.3M/year
- Top 100 average $7M/year
- Uses Lua scripting — Claude Code can accelerate development dramatically

## Most Lucrative Game Types
1. **Tycoon games** — business simulation with progression
2. **Obbies** (obstacle courses) — with cosmetic progression
3. **Social hangout spaces** — with cosmetic monetization
4. **Mini-game collections** — variety keeps players engaged

## Getting Started
1. Download Roblox Studio (free): https://create.roblox.com/
2. Learn basics of Roblox Lua scripting
3. Use Claude Code to rapidly prototype game mechanics
4. Publish, test engagement, iterate

## Revenue Model
- In-game purchases via Robux (Roblox currency)
- Developer Exchange (DevEx) converts Robux to USD
- Need 100k+ earned Robux to cash out
- Roblox takes ~75%, you keep ~25% of gross revenue

- Start simple, ship fast, iterate based on player data
